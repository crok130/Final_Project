## 💡 Motivation
팀원들과의 회의를 통해 일회성 및 당근 마켓과 같은 중고거래 플렛폼 사이트를 만들고자 했습니다.  <br/>
타인과의 거래가 복잡해지는 세상. 익숙하지 않은 사람과 중고 거래를 하는 것은 부담이 따를 수 있기에, <br/>
우리는 간편하고 안전한 거래 환경을 구축하기로 했습니다. <br/>
사람들이 믿고 거래할 수 있는 플랫폼. 그것이 저희의 프로젝트입니다.
